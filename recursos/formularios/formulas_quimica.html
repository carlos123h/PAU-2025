Estructura atómica

Ecuación de Planck
E = h · ν
Donde h = 6,626·10-34 J·s (constante de Planck)
Longitud de onda de De Broglie
λ = h / (m·v)
Relaciona las propiedades ondulatorias y corpusculares
Energía del fotón
E = h·c / λ
c = 3·108 m/s (velocidad de la luz)
Principio de exclusión de Pauli
Dos electrones en un átomo no pueden tener los 4 números cuánticos iguales

Termodinámica química

Primer principio de la termodinámica
ΔU = Q + W
ΔU = variación de energía interna, Q = calor, W = trabajo
Relación entre ΔH y ΔU
ΔH = ΔU + Δ(PV)
Para gases ideales: ΔH = ΔU + ΔnRT
Ley de Hess
ΔHreacción = Σ ΔHf(productos) - Σ ΔHf(reactivos)
Permite calcular ΔH a partir de entalpías de formación
Variación de entalpía
ΔH = Qp
A presión constante, ΔH equivale al calor intercambiado

Cinética química

Velocidad de reacción
v = -1/a · d[A]/dt = 1/b · d[B]/dt
Para la reacción aA → bB
Ecuación de velocidad
v = k · [A]α · [B]β
k = constante de velocidad, α y β son los órdenes parciales
Ecuación de Arrhenius
k = A · e-Ea/RT
Relación entre k y la temperatura
Tiempo de vida media (1er orden)
t1/2 = ln(2)/k
Tiempo para que la concentración se reduzca a la mitad

Equilibrio químico

Constante de equilibrio Kc
Kc = [C]c[D]d / [A]a[B]b
Para la reacción aA + bB ⇄ cC + dD
Constante de equilibrio Kp
Kp = (PC)c(PD)d / (PA)a(PB)b
En función de presiones parciales
Relación entre Kc y Kp
Kp = Kc · (RT)Δn
Donde Δn = (c+d) - (a+b)
Producto de solubilidad
Kps = [Ax+]a · [By-]b
Para la sal AaBb

Equilibrio ácido-base

Constante de acidez
Ka = [H+][A-] / [HA]
Para un ácido HA
Constante de basicidad
Kb = [BH+][OH-] / [B]
Para una base B
Producto iónico del agua
Kw = [H+][OH-] = 10-14
A 25°C
pH y pOH
pH = -log[H+]
pOH = -log[OH-]
pH + pOH = 14
Relación entre Ka y Kb
Ka · Kb = Kw
Para un par ácido-base conjugado
pH en el punto de equivalencia
pH = 7 (ácido fuerte + base fuerte)
pH > 7 (ácido débil + base fuerte)
pH < 7 (ácido fuerte + base débil)

Reacciones redox

Potencial estándar de celda
E°celda = E°cátodo - E°ánodo
También: E°celda = E°reducción - E°oxidación
Ecuación de Nernst
E = E° - (RT/nF) · ln Q
A 25°C: E = E° - (0,059/n) · log Q
Relación entre ΔG° y E°
ΔG° = -nFE°
F = 96485 C/mol (constante de Faraday)
Leyes de Faraday para la electrólisis
m = (M · I · t) / (n · F)
m = masa depositada (g)
M = masa molar (g/mol)
I = intensidad de corriente (A)
t = tiempo (s)
n = número de electrones intercambiados
F = constante de Faraday (96485 C/mol)

Química orgánica

Tipos de isomería
Isomería estructural Estereoisomería
De cadena Diferentes esqueletos carbonados Geométrica Cis-Trans (Z-E)
De posición Diferente posición del grupo funcional Óptica Enantiómeros (quiralidad)
De función Diferentes grupos funcionales
Reacción de adición
C=C + HX → C-C-X
X puede ser Cl, Br, OH, etc.
Reacción de sustitución
R-H + X2 → R-X + HX
X2 puede ser Cl2, Br2, etc.
Reacción de eliminación
C-C → C=C + H2O
Ejemplo: deshidratación de alcoholes
Reacción de condensación
R-COOH + R'-OH → R-COO-R' + H2O
Ejemplo: esterificación
